Why Context Kit?

AI coding assistants struggle when you feed them incomplete context. A file at a time is rarely enough — bugs hide in the seams between modules, refactors ripple across the call graph, and feature work needs to touch multiple layers. But dumping entire files burns through the context window and drowns the model in noise.

Context Kit lets you hand-pick the relevant slices, tag them by intent, and combine them into a single coherent prompt. It is a small browser-side tool for assembling prompt context deliberately, one snippet at a time.

Inspired by Ask HN threads on improving the AI coding loop — "AI coding is a nightmare" and "experimenting with LLMs for coding".
