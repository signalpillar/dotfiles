Prototypes

Small self-contained HTML prototypes published as static GitHub Pages files.

Jump Rope Interval Timer Alternates work and break timers, tracks completed rounds, and keeps controls large for mobile use. Browser LLM Chat Loads a small WebGPU model in the browser and provides a simple chat surface with streaming replies. Meeting Cost Meter Turns attendee count and hourly rate into a live money burn meter for ending low-value meetings faster. Context Kit Assemble and manage prompt context for AI coding sessions: paste snippets, estimate tokens, tag by intent, drag to reorder. Noise Nudge Uses the microphone locally to show live room loudness and count distracting peaks over a threshold.
